>文档类型声明标签
```
1.文档类型声明：作用就是告诉浏览器使用哪种HTML版本来显示网页
2.<!DOCTYPE html>:这句话的意思当前页面采取的是HTML5版本来显示网页
3.<!DOCTYPE html>:声明位于文档中的最前面的位置，处于<html>标签之前
4.<!DOCTYPE html>:不是一个HTML标签，它就是文档类型声明标签

```




>语言种类属性
```
1. 属性值  en  定义语言为英文
2. 属性值  zh-CN  定义语言为中文
3. 属性值  fr  定义语言为法语






1. lang  属性用来定义当前文档显示的语言

2. 简单来说，定义属性 en 就是英文网页，定义属性zh-CN 就是中文网页

3. 其实对于文档显示来说，定义成 en 的文档也可以显示中文定义 zh-CN 的文档也可以显示英文

4. 这个属性对浏览器和搜索引擎（百度，谷歌）还是有作用的

5. 当我们使用 en 将网页定义为英文网页，但是我们在 body 还是使用中文，所以再打开浏览器的时候进行刷新则会提示我们需不需要将网站翻译为英文

6. 而现在我们制作的网页都是针对中国，所以一般还是使用 zh-CN ，
比如京东网站则使用的就是 zh-CN 

```




> 字符集


```
1. 字符集（Charachter set）则就是多个字符的集合以便计算机能够识别和存储各种文字

2. 在`<head>` 标签内，可以通过`<meta>` 标签的`charset` 属性来规定HTML文档应该使用哪种字符编码

3. `charset` 属性常用的属性值有：`GB2312` ，`BIG5` ，`GBK` ，和`UTF-8` ，其中`UTF-8` 也被称为**万国码**，基本包含了全世界所有国家需要用到的字符。

4. 上面语法是必须要写的代码，否则可能引起乱码的情况，一般情况下，统一使用`UTF-8 ` 编码，尽量统一写成标准的`UTF-8` 
```
