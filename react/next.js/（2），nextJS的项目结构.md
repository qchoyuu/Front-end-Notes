>nextJS的项目结构
```
顶级文件
   1.next.config.js -> next.js的配置文件
   2.middleware.ts -> next.js请求中间件
   3..env -> 环境变量
   4..env.loval ->本地环境变量
   5..env.development -> 开发环境变量
   6..next-env.d.ts -> next.js中typeScript声明文件
   7.package.json -> 项目依赖和脚本
   8..gitignre -> 要忽略的git文件和文件夹
   9.tsconfig.json -> TS配置文件
   10.jsconfig.json -> JS配置文件
   11.eslintrc.json -> ESLint配置文件


顶级文件夹
   1.app -> 应用路由器
   2.page -> 页面路由器
   3.public -> 静态资源
   4.src -> 可选的应用程序源文件夹


```
